# Lava

# Introduction

## Dependencies
 - GLFW for window system
 - GLM for matemathics
 - STBI for texture reading
 - ASSIMP for meshes loading

## Building
Lava is based on Cmake.

```sh
$ git clone https://github.com/maldicion069/lava.git
$ cd lava
$ mkdir build && cd build
$ cmake .. -DCMAKE_BUILD_TYPE=Release
$ make
```
